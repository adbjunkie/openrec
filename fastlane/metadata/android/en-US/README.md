# Store metadata

F-Droid can use these files for the app listing.

## Screenshots (add before submitting)

Capture on a device and place PNGs here:

```
fastlane/metadata/android/en-US/images/phoneScreenshots/
  01_main.png
  02_recording.png
  03_stealth.png
```

Recommended size: 1080×1920 or similar phone aspect ratio.
